# Tiny Examples

This folder contains small, focused examples for learning and reference.

## Purpose
- Quick demonstrations of concepts
- Learning materials separate from production code
- Easy to run and understand
- Can be referenced anytime

## Contents

### 01_toy_example_log_transform.py
- Demonstrates log transformation for non-negative predictions
- Shows step-by-step how log1p/expm1 works
- Compares regular vs log-transformed regression

## Usage
Run any example directly:
```bash
python toy_example_log_transform.py
```
